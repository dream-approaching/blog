### 通用

#### 重命名文件

```sh
# 将file1.txt重命名为file2.txt
mv file1.txt file2.txt
```

### Linux

#### 查看系统版本

```sh
cat /etc/issue
# Ubuntu 20.04.1 LTS \n \l

lsb_release -a
# No LSB modules are available.
# Distributor ID: Ubuntu
# Description:    Ubuntu 20.04.1 LTS
# Release:        20.04
# Codename:       focal
```
